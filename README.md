# Rutgers Physics & Astronomy — Department Signage

A Raspberry Pi kiosk that displays department events as a full-screen slideshow on a TV. Data is pulled automatically from the weekly newsletter and department website.

## What it shows

- **Colloquium** — speaker, title, location (abstract if Playwright is installed)
- **Seminars** — all weekly seminars grouped by day (CMS, HEX, HET, MPS, IQB, SPS, etc.)
- **Irons Public Lecture** — upcoming special lecture with poster image
- **Department News** — recent faculty awards and announcements
- **Next week preview** — colloquium and seminars for the following week
- **QR codes** on each slide linking to the relevant department page
- **News ticker** at the bottom with recent department news

## Data sources

| Source | URL | Notes |
|--------|-----|-------|
| Weekly newsletter | physics.rutgers.edu/newsletter | Primary source, no login required |
| Irons lecture | physics.rutgers.edu/irons | Poster image + event details |
| Department news | physics.rutgers.edu/news/2026-news | Recent announcements |
| Colloquium page | physics.rutgers.edu/events/colloquium | Requires Playwright for abstracts |
| Seminars page | physics.rutgers.edu/events/seminars | Requires Playwright for abstracts |

## Raspberry Pi setup (one time)

Tested on **Raspberry Pi OS with Desktop (Bookworm)**.

```bash
git clone https://github.com/b0burk02/physics-signage.git ~/physics-signage
cd ~/physics-signage
bash setup.sh
```

The setup script:
- Installs Python dependencies and Playwright
- Generates the initial `output/display.html`
- Installs an hourly cron job to refresh data
- Configures Chromium to launch in kiosk mode on desktop login

Reboot the Pi after running setup. The display will start automatically.

## Manual update

```bash
cd ~/physics-signage
venv/bin/python update.py
```

## Optional: colloquium abstracts

Abstracts require Playwright (a headless browser) to bypass the department website's bot-protection. The setup script installs it automatically. If it was skipped or failed:

```bash
venv/bin/pip install playwright
venv/bin/python -m playwright install chromium
venv/bin/python -m playwright install-deps chromium
```

Then re-run `update.py` — the colloquium slide will show the full abstract.

## Two TVs

The Raspberry Pi 4 and 5 have two micro-HDMI ports. To enable both:

Add to `/boot/firmware/config.txt`:
```
hdmi_force_hotplug:0=1
hdmi_force_hotplug:1=1
```

Then add this line to the kiosk autostart (before the Chromium line) in
`~/.config/lxsession/LXDE-pi/autostart`:
```
@xrandr --output HDMI-1 --same-as HDMI-2
```

## Updating the code

On your development machine:
```bash
git add -A && git commit -m "describe your change" && git push
```

On the Pi:
```bash
cd ~/physics-signage && git pull && venv/bin/python update.py
```

## File structure

```
physics-signage/
├── update.py        # Main script: scrapes data and generates display.html
├── setup.sh         # One-time Raspberry Pi setup
├── requirements.txt # Python dependencies
└── output/          # Generated by update.py (not in git)
    ├── display.html # The TV display page
    ├── events.json  # Cached event data
    └── irons_poster.jpg
```

## Cron schedule

Data refreshes every hour automatically. To change the frequency, edit the crontab:

```bash
crontab -e
```
